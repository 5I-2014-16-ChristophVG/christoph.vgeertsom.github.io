Taak 03 Vertaling van een instructiefilmpje

De opgave van deze taak was om een instructiefilmpje online te gaan zoeken, nadat we het filmpje hadden gevonden moesten wij de tekst die werd gebruikt opschijven in mooi Engels. Na het in het mooi Engels schrijven moesten we dat vertalen in het Nederlands. Voor te vertalen moesten we de zinnen zo kort mogelijk houden en ook zien dat het begrijpbaar Nederlands was. Nadat we de tekst hadden vertaald moesten we dit op het eigenlijke echte instructiefilmpje zetten. In het filmpje kan u zien welke programma’s ik heb gebruikt om te vertalen.
Onderaan kan u de de uitgetypte tekst in het Engels en het Nederlands zien.

Hier kan u het ondertiteld filmpje vinden

Engelse tekst

Hello, and welcome to “beginners guide to Linux” by Shadowformed Productions. If you came across this video you have probably heard Linux on the internet or from friends that use Linux but you don’t exactly know what it is. Let’s assume you’re not swift with computers, when you turn on your pc, chances are it’s running Windows or Mac OS X. Microsoft Windows and Mac OS X are examples of an operating system. Simply an operating system is the software that operates your computer. Windows has a taskbar which you use to run your applications, a file browser that you use to manage your files like photos and music, a web browser to access the internet and more. Linux, officially known as GNU/Linux, is another operating system with its own desktop, file browser and many different software applications, so the question is if Linux does everything that Windows does, then why would I use Linux as my computer’s operating system? There are a few key differences between Linux and other operating systems like Windows and Mac OS.

Linux is open source software which basically means that anyone can change the operating system. if someone doesn’t like the way a part of Linux works, they are able to change it to a way they like to suit their needs. Because of this particular type of licensing, you see many different versions of Linux or distributions available. Each of them are similar but with their own benefits and all are Linux. This means that you have the freedom of choice to find a version of Linux that works best for you and believe me, there is a distribution out there for everyone. While operating systems like Windows or Mac OS may suit your needs just fine. If there’s something about the way it works that bothers or annoys you, there is very little chance of it being changed or improved.

The second big difference is cost, currently Windows 8 Pro runs for forty dollars at it’s the cheapest. Mac OS Mountain Lion runs for twenty. Both are much cheaper than they used to be, you have always been able to download most distributions of Linux completely for free. With software improvements and bug fixes to Linux offered free as well. Application software is the same story. Almost all software for Linux is free, from office suites to web applications, games and even photo and video editors.

Nederlandse tekst

Hallo en welkom bij deze beginnersgids voor Linux. Als je deze video tegenkomt, heb je vast al gehoord van Linux, op het internet of van vrienden die Linux gebruiken maar je weet niet exact wat het is. Laten we ervan uitgaan dat je niet goed overweg kan met computers, als je jouw computer aanzet is de kans groot dat het Windows of Mac OS X draait als besturingssysteem. Een besturingssysteem is software dat je computer bestuurt. Windows heeft een taakbalk waar je je applicaties op kan starten, een verkenner waarmee je je bestanden zoals foto’s en muziek kan beheren, het heeft ook een webbrowser om op het internet te surfen en nog veel meer. Linux, officieel eigenlijk GNU/Linux, is een ander besturingssysteem met z’n eigen desktop, verkenner en nog vele andere software applicaties. Dus als Linux hetzelfde doet als Windows, waarom zou ik dan Linux gebruiken als het besturingssysteem van mijn computer? Er zijn een paar belangrijke verschillen tussen Linux en andere besturingssystemen zoals Windows en Mac OS.

Linux is een vrije software wat eigenlijk wil zeggen dat iedereen het besturingssysteem kan aanpassen. Als iemand niet leuk vindt hoe Linux werkt, is het mogelijk om dit aan te passen. Door het toepassen van deze bepaalde type licensie zie je veel verschillende versies van Linux of distributies die beschikbaar zijn. Ze zijn allemaal ongeveer gelijk aan elkaar maar met hun eigen voordelen en ze zijn allemaal Linux. Dit betekent dat je volledige vrijheid hebt voor het kiezen van een versie van Linux die het beste werkt voor jou. Besturingssystemen zoals Windows en Mac OS kunnen voldoen aan jouw behoeften. Als er iets is dat jou stoort of verveelt, is er weinig kans dat het verandert of verbetert wordt.

Een ander groot verschil is de prijs, Windows 8 Pro werkt al voor 40 dollar. Mac OS Mountain voor 20 dollar. Beide zijn veel goedkoper dan dat ze vroeger waren. Je hebt echter altijd al verschillende distributies van Linux volledig gratis kunnen downloaden. Inclusief software verbeteringen en bug verbeteringen door Linux aangeboden. Applicaties zijn hetzelfde verhaal, bijna alle software in Linux is compleet gratis, van kantoortoepassingen tot web applicaties, spelletjes en zelfs foto en video editors.
